**[üöÄ Live Demo](https://Sayandeepghosh.github.io/ai_stock_predictor/)**

---

## ‚ú® Features

*   **Real-Time Data**: Fetches live stock prices and historical data for **US (NASDAQ/NYSE)** and **Indian (NSE/BSE)** markets via Yahoo Finance.
*   **AI-Powered Predictions**:
    *   **Trend Analysis**: Uses RSI, Moving Averages (SMA), and Volatility to determine market sentiment.
    *   **Future Forecasting**: Generates 7-day price projections using deterministic random walk models grounded in technical indicators.
    *   **Confidence Score**: Provides a confidence metric for every prediction.
*   **Smart Search**:
    *   Dynamic autocomplete for thousands of global stocks.
    *   Supports searching by Ticker (`AAPL`, `RELIANCE.NS`) or Company Name (`Tesla`, `Zomato`).
*   **Interactive Charts**:
    *   Professional-grade candlestick charts using `lightweight-charts`.
    *   Visualizes historical data alongside future predicted trends.
*   **Investment Calculator**: Built-in simulator to estimate potential returns based on predictions.
*   **Production Ready**:
    *   Frontend hosted on **GitHub Pages**.
    *   Backend hosted on **Hugging Face Spaces** (Dockerized).
    *   Automatic fallback to client-side analysis if the backend is unreachable.

## üöÄ Getting Started

### Prerequisites
*   Node.js (v16+)
*   Python (v3.10+)
*   Git

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Sayandeepghosh/ai_stock_predictor.git
    cd ai_stock_predictor
    ```

2.  **Setup Backend**
    ```bash
    cd backend
    pip install -r requirements.txt
    python main.py
    ```
    *The backend will start at `http://localhost:8000`*

3.  **Setup Frontend** (Open a new terminal)
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
    *The frontend will start at `http://localhost:5173`*

## üåç Deployment Architecture

This project uses a hybrid deployment strategy to ensure high availability and zero cost:

1.  **Frontend**: The React app is built as a static site and deployed to **GitHub Pages**.
2.  **Backend**: The FastAPI application is containerized using **Docker** and deployed to **Hugging Face Spaces**.
3.  **Connectivity**: The frontend automatically detects the environment.
    *   **Localhost**: Connects to `http://localhost:8000`.
    *   **Production**: Connects to the Hugging Face Space URL.
